desconstroi #

Arguments

:: Mapa

Mapa atual.

-> Instrucoes

Mapa desconstruido em forma de Instrucoes.

Desconstrói um Mapa numa sequência de Instrucoes.

NB: Uma solução correcta deve retornar uma sequência de Instrucoes tal que, para qualquer Mapa válido m, executar as instruções '(desconstroi m)' produza o mesmo Mapa m.

NB: Uma boa solução deve representar o Mapa dado no mínimo número de Instrucoes, de acordo com a função tamanhoInstrucoes.
